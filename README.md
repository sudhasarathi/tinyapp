## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- method-Override
- cookie-session
- mocha (for tests)
- chai (for tests)

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.

## How to Use

- Run your node server
- Browse to http://localhost:8080/
